# Fusion Helper Installer

An automated, user-friendly installer for **Fusion Helper**, packaged using the Nullsoft Scriptable Install System (NSIS). This project provides a streamlined installation experience, ensuring all necessary components are placed correctly for immediate use.

* **Uninstaller Included:** Cleanly remove the tool and its components whenever needed.
